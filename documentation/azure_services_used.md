# Azure Services Used

The following Azure services were used in the project:

- **Azure Virtual Network** — network segmentation
- **Subnets** — isolation of VM workloads
- **Network Security Groups (NSG)** — traffic filtering
- **Public IP** — temporary access for testing
- **Network Watcher** — NSG Flow Logs and diagnostics
- **Azure Bastion (planned)** — secure remote access
- **Azure Backup** — protection of VM data
